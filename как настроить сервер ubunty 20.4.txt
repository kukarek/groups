Для настройки Ubuntu 20.04 для работы с Python 3.9+ и указанными библиотеками, выполните следующие шаги:

Шаг 1: Обновление системы
Откройте терминал и выполните следующие команды, чтобы обновить список пакетов и установить доступные обновления:

  sudo apt update
  sudo apt upgrade

Шаг 2: Установка Python 3.9+
Python 3.9 может не входить в стандартный репозиторий Ubuntu 20.04, поэтому придется добавить репозиторий deadsnakes:

  sudo apt install software-properties-common
  sudo add-apt-repository ppa:deadsnakes/ppa
  sudo apt update
  sudo apt install python3.9

После установки можно проверить версию Python:

  python3.9 --version

Шаг 3: Установка необходимых пакетов и библиотек
Установите необходимые библиотеки с помощью pip, который, как правило, устанавливается автоматически вместе с Python:

  sudo apt install python3.9-distutils
  apt install curl
  curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
  sudo python3.9 get-pip.py

Шаг 4: Создание виртуальной среды (рекомендуется)
Хорошей практикой является создание виртуальной среды для каждого проекта, чтобы изолировать зависимости. Вам понадобится пакет venv:

  sudo apt install python3.9-venv

Затем создайте виртуальную среду в папке вашего проекта:

  python3.9 -m venv myenv

Активируйте виртуальную среду:

  source myenv/bin/activate

После завершения работы с виртуальной средой, вы можете её деактивировать с помощью команды deactivate.

Теперь ваша Ubuntu 20.04 настроена для работы с Python 3.9+ и указанными библиотеками. Вы можете создать и запускать свои проекты, используя этот окружение.
